# Epic-Arpeggiator  
**TFYA65. Grupparbete i ljudfysik**

## Om projektet
Epic-Arpeggiator är ett grupparbete inom kursen TFYA65. Projektet är en webbaserad applikation för ljudsyntes där grundläggande ljudfysik används för att skapa och manipulera ljud i realtid. Applikationen kombinerar ljud, visualisering och interaktion för både lärande och kreativt skapande.

## Funktioner
- Sinusbaserad ljudsyntes  
- Övertoner och undertoner  
- Vibrato och tremolo  
- Ekoeffekt  
- Arpeggiator med tempo, gate, oktav och spelordning  
- Realtidsvisualisering av ljudvågor  

## Hur man använder
1. Klona eller ladda ner repositoryt  
2. Öppna `index.html` i en modern webbläsare  
3. Justera frekvens och amplitud för grundtonen  
4. Aktivera önskade ljudeffekter och arpeggiator  
5. Använd tangentbordet för att spela toner och skapa arpeggios  

Ingen installation krävs. Allt körs lokalt i webbläsaren via Web Audio API.

## Teknik
- JavaScript  
- HTML och CSS  
- Web Audio API  

## Grupp
Projektet genomfördes i grupp inom kursen TFYA65.
